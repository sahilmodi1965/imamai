# imamai - Islamic Guidance Web App

An Islamic guidance application that helps Muslims find answers to their questions using authentic Islamic sources as the foundation.

## Project Vision

imamai aims to provide reliable Islamic guidance by leveraging authentic books and scholarly sources. Users can ask questions and receive answers grounded in traditional Islamic knowledge.

## Tech Stack

- **Framework:** Next.js 15 with App Router
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Deployment:** TBD

## Target Users

Muslims seeking authentic Islamic guidance and answers to their questions about faith, practice, and daily life.

## Getting Started

First, install dependencies:

npm install

Then, run the development server:

npm run dev

Open http://localhost:3000 with your browser to see the result.
